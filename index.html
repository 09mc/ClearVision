ClearVision

ClearVision is an automatically updating, easily customizable Theme for Discord.

ClearVision is built for use with BetterDiscord.

For update notifications and theme support, join the ClearVision Discord server.

If you would like to contribute, check out the Github Repository.

ClearVision v5

This is the Theme version of ClearVision for BetterDiscord.

If you wanna download the original Theme, head over to the Github Releases and download the Theme of your choosing. You can find installation instructions in the ClearVision Wiki.

ClearVision v6

This project is still work-in-progress currently. The goal is to rework ClearVision as an even easier to use and more user-friendly Plugin for BetterDiscord.

Participation in the development process is currently limited to Contributors, Donators and Testers.

If you would like to participate in the future, make sure to join the ClearVision Discord server.
