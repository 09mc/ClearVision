Jiiks - 02/21/2017Reply

@everyone There's been some rumours about BetterDiscord stealing your token or other users being able to steal your token. While it is possible for plugins to steal your token; as long as you don't install plugins from unknown source you're fine. BetterDiscord in itself does not access your token and certainly does not submit it anywhere. Security of users is very important to me and I'm currently writing v2 of BetterDiscord that blocks unauthorized scripts from running at all outside developer mode(it also blocks any attempts of getting your token through "hacks/exploits"). Please do not blindly add plugins without verifying the source code. tldr; don't install dodgy plugins and everything is fine.

Jiiks - 02/21/2017Reply

People claim that tokens are not tied to your system, in that case it's a faulty token system that we can do nothing about.

Jiiks - 02/24/2017Reply

Guess I'll make this about the cloudflare "big data leak" No there was no "big data leak". The chances of anyone getting access to any of your information are extremely tiny. What happened is that in some edge cases cloudflare would print possibly private information from memory due their to autocompiler using == instead of >=. Some private data was then cached by search engines and maybe someone somehow found someone's dog's name. This was already fixed ages ago before anyone besides cloudflare/google/etc even knew about it.
